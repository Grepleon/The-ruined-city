# Разрушенный город
Моя игра - это довольно старая идея, решил реализовать на python
(Далее здесь будет описание игры, однако также игра описана в разделе кнопки "Помощь")

"Разрушенный город" — это игра про выживание в процедурно генерируемом мире с ограниченной видимостью. Игрок исследует клеточную карту, сражается с врагами и управляет температурой окружающего пространства.

Основные механики:
  Ограниченный обзор + фонарик и спички для освещения
  Температурная система: холод наносит урон, вода замерзает, земля покрывается снегом
  Боевая система с атакой мечом по направлению к курсору
  14 сюжетных уровней и 7 режимов выживания
  9 типов врагов с разным поведением (3 из них — боссы) *1
  Игрок использует WASD для движения, мышь для атаки, пробел для спичек и E для фонарика, чтобы выживать в постепенно усложняющихся условиях.

*1: Враги в игре:

  Обычные враги:
    Холод - неподвижный элементальный враг, понижает температуру вокруг. Появляется только на земле. Уязвим к высоким температурам.
    Леденящий - аналогичен Холоду, но появляется только на воде.
    Рыцарь - бронированный воин, преследует игрока по кратчайшему пути *2. Не может ходить по воде. Ходит раз в два такта *3. Наносит урон при близком контакте. Умирает, если его ударить мечом.
    Амфибия - водное существо, может перемещаться по суше и воде. По воде движется в два раза быстрее. В остальном аналогичен Рыцарю.
    Маг - дальнобойный противник, заряжает магические атаки (расширяющиеся круги). Не может ходить по воде, только по суше. Появляется только на суше.
    Призрак - проходит сквозь любые препятствия. Наносит урон при близком нахождении. Боится света фонарика.
  Боссы:
    Мега-Рыцарь (уровень 6) - состоит из четырех частей *4. Может "затаптывать" воду в землю. Атакует по осям. Уязвим только к ключу.
    Гидра (уровень 12) - превращает землю в воду. Аналогично уязвима только к ключу.
    Некромант (уровень 14) - мощный маг, способен к телепортации. Наносит урон как Мега-Рыцарь. Уязвим только к мечу.
  Предмет:
    Ключ - специальный предмет для нанесения урона боссам. Активируется только на определенных условиях в соответствующих уровнях.

*2: Кратчайший (почти) поиск пути был сделан с помощью алгоритма A-star (алгоритм A *). Эвристика была умножена на коэффицент 1.5, чтобы быстрее находился путь (взамен на точность)
*3: Реализация хода раз в два такта реализовано через проверку четности данного такта
*4: Реализация четырехклеточности босса послужило централизованная система координат для босса (то есть координаты босса вынесены в отдельную переменную)
